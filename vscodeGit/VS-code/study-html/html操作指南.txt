<!--
    1 标题   h1 - h6 且从大到小
      段落   p标签
      换行   br 直接换行 / hr 换行并带一个分割线
    2 列表   ol：有序列表（可嵌套）
           ul：无序列表（可嵌套）
           li：列表项（有序/无序都用） 
    3 超链接 a 
              herf    用于定义跳转目标资源的地址
                    完整url "http://www.atguigu.com"
                    相对路径 以当前资源的所在路径为出发点找目标资源
                            ./ 表示当前资源所在的路径，可以省略不写
                            ../表示当前路径的上一层，需要时必须写出
                    绝对路径 无论资源在哪，使用固定的位置作为出发点去找目标资源
              target  用于定义目标资源的打开方式
                      _self  在当前窗口打开目标资源
                      _blank 在新窗口打开目标资源
    4 图片 img 
              scr   用于定义图片的连接
              title 用于定义鼠标悬停时显示的文字
              alt   用于定义图片加载失败时显示的提示文字
    5 表格 table 整张表格
           thead 表头（可写可不写）
           tbody 表体
           tfoot 表尾
                tr 表格中的一行
                    td 行中的一个单元格
                    th 自带加粗和剧中的td
            rowspan="n" 加在td里，向下侵占n格
            colspan="n" 加在td里，向右侵占n格
    6 表单 form 其内部用于定义让用户输入信息的表单项标签
              action form属性，用于定义信息提交的服务器地址
                  url/相对路径/绝对路径
              method form属性，用于定义信息的提交方式
                  get方式 1.数据会缀到url后，以？作为参数开始的标识，多个参数用&隔开 形式url？key=value&key=value&key=value
                          2.数据直接暴露在地址栏上，相对不安全
                          3.地址栏长度有限，所以提交的数据量不大
                          4.地址栏上，只能是字符，不能提交文件
                  post方式，数据会通过请求体发送，不会缀到url后
                          1.参数默认不放在url后
                          2.数据不会直接暴露在地址栏上，相对安全
                          3.数据是单独打包通过请求体发送，提交的数据量比较大
                          4.请求体中，可以是字符，也可以是字节数据，可以提交文件
                          5.相较于get效率略低
            表单项标签 1.表单项标签一定要定义name属性，该属性用于明确提交时的参数名 
                      2.还需要定义value属性，该属性用于明确提交时的实参
            input
                type 输入信息的表单项类型
                    text     单行普通文本框
                    password 密码框
                    submit   提交按钮
                    reset    重置按钮  
                    radio    单选框
                             多个单选框使用相同的name属性值，则会有互斥效果；如选择男女时两个input里面的name必须相同           
                             checked  默认自动选中此选项
                    checkbox 多选框，同时可以选中多个
                    file     可以拖拽文件进行上传
                    hedden   隐藏域，不显示在页面上，提交时会展示  
            textarea 文本域，显示多行文字，相当于一个下拉框
            select   下拉框
                    option 默认选中的情况下可以添加selected
    7 布局   css样式设置，通过style属性进行设置：style="样式名：样式值；样式名：样式值：......"
                   块元素： 自己独占一行的元素 块元素的css样式的宽高一般都是生效的
                        div 
                   行元素： 自己不会独占一行的元素 行内元素的css样式宽高一般都是不生效的
                        span
    8 转义字符 空格：&nbsp
              <   : &lt
              >   : &gt
              &   : &amp
